Concept

Grab'n slide doesn't add any html to your code. You build your own slider, the librairy make it draggable. That's it.

Options

the following * are mandatory

container*

Represent the draggued container. Its margin-left will change on drag (don't forget to add a will-change balise on this element).

new Grabnslide({
  container: document.querySelector('#slider ul')
})

cell

If the cell option isn't specified, the container's full width will be used to calculate the container's margin range (to avoid being out of window).

If the cell option is specified, the container's margin limit will be the sum of all cells's width (can be more accurate, especially if you work with flex-box).

new Grabnslide({
  container: document.querySelector('#slider ul'),
  cell: document.querySelectorAll('#slider ul li')
})

triggerElement

The triggerElement is the clicked making the container moves. If it isn't specified, the container will be used as trigger.

new Grabnslide({
  container: document.querySelector('#slider ul'),
  triggerElement: document.querySelector('#slider #sidebar')
})
